# PySyMex Benchmarks

This directory contains benchmark data used to track analysis performance across releases.

## Running Benchmarks

```bash
# Run the full benchmark suite
pysymex benchmark --format json -o benchmarks/baseline.json

# Compare against a previous baseline
python scripts/ci_perf_gate.py --baseline benchmarks/baseline.json

# Human-readable markdown output
pysymex benchmark --format markdown
```

## Files

- `baseline.json` — generated by running the benchmark suite on the current release
- `README.md` — this file

## What is measured

- **Paths explored per second** — symbolic execution throughput
- **Z3 solve time** — constraint solving latency
- **Total analysis time** — end-to-end per-file scan time
- **Memory usage** — peak RSS during analysis

## Generating a new baseline

```bash
pysymex benchmark --format json -o benchmarks/baseline.json
```

Then commit the updated `baseline.json` as part of the release commit.
